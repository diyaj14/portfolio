# Diya Joshy — Portfolio (Futuristic, Framer Motion)

This is a ready-to-run React + Vite portfolio template built for **Diya Joshy — Creative Developer**.

## Features
- Single-scroll page with sticky navbar
- Framer Motion animations
- TailwindCSS for styling (config included)
- Sections: Hero, About, Projects, Contact, Footer

## To run locally
1. unzip the folder and open a terminal in the project root
2. install dependencies:
```bash
npm install
```
3. start dev server:
```bash
npm run dev
```
4. open the printed local dev URL (usually http://localhost:5173)

## Notes
- Tailwind is configured; if you want to customize fonts/colors edit `tailwind.config.cjs` and `src/index.css`.
- Replace project details in `src/data.js` or directly inside components.
